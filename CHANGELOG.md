## [Unreleased]

### Added
- Initial release of MCHEP.
- Core Monte-Carlo integration algorithms (VEGAS, VEGAS+).
- SIMD acceleration for integration grids.
- MPI support for distributed integration.
- GPU acceleration support via `burn`.
- C-API (`mchep_capi`) for C/C++ interoperability.
- Python bindings (`mchep_pyapi`) for high-level usage.
- Extensive documentation and benchmarks.
- CI/CD workflows for automated testing and releases.
